Map es una coleccion de pares clave valor, donde cada clave esta asociado a un unico valor.
- Map no puede tener claves duplicadas.
- Permite acceso rapido a travez de sus claves.

Metodos:

1. put(clave,valor) = Agregar un par clave-valor a la coleccion map.
2. get(object clave) = Devuelve el valor asociado a la clave brindada o null si la clave no existe.
3. containsKey(object clave) = Verifica si el Map contiene la clave especficada.
4. containsValue(object valor) = Comprueba si el Map contiene el valor especficado.
5. remove(object clave) = elimina el registro asociado a la clave brindada.
6. keySet() = Devuelve un conjunto de todas las claves en el map.
7. values() = Devuelve una coleccion de todos los valores que contiene el map.
